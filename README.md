# rumpeg
A WIP MPEG-1 Encoder written in Rust. Bugs may still exist.

### COMPLETED:
- 2D Approximate 8x8 DCT Algorithm
- Quantization Algorithm
- Zig-Zag Encoding Algorithm
- I-Frame Block Encoding
- I-Frame Macroblock Encoding
